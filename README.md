# Simple Database API


## This is simple Store database that stores and retrieves objects from the file system.

## It has the following API methods:
    * create()
    * findById()
    * find()
    * findByIdAndUpdate()
    * findByIdAndDelete()

## It uses the the following Node.js filesystem methods:
    * fs.writeFile
    * fs.readFile
    * fs.readdir
    * fs.unlink
